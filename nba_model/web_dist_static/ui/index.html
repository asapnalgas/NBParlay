Local Prediction Engine

NBA projections built for fast decision-making, not spreadsheet hunting.

Starter-first board with live sync, injury context, lineup context, and clear guardrails for what is actionable vs what should be a pass.

Board style Starter-first player cards
Core markets PTS, REB, AST, PRA, 3PM, STL, BLK, TOV
Live behavior Auto-refresh + in-game updates

Public mode is active. Advanced data/model controls are hidden.

Player Projection Board

PrizePicks-style player cards with starter-first ranking, playable-default filtering, pregame anchors, confidence/error bands, and line cross-reference when imported.
